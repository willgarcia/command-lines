docker - list images
--------------------

```shell
docker images
```

docker - rm image
-----------------

```shell
docker rmi -f [image-id]
```

docker - list containers
------------------------

```shell
docker ps
```

docker - rm container
---------------------

```shell
docker rm -f [container-id]
```

docker - execute a command 
--------------------------

```shell
docker exec -i -t [container-id] bash
```

fig - remove containers
-----------------------

```shell
fig rm --force
```
