shortcuts - halt a command
--------------------------

```shell
ctrl-c
```

shortcuts - stop a command
--------------------------

```shell
ctrl-z
```

shortcuts - exit
----------------

```shell
ctrl-d
```

shortcuts - erase last command word
-----------------------------------

```shell
ctrl-w
```

shortcuts - erase whole line
----------------------------

```shell
ctrl-u
```


shortcuts - search history
--------------------------

```shell
ctrl-r
```
