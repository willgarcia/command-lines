screen - list
-------------

```shell
screen -l 
```

screen - attach last screen
---------------------------

```shell
screen -rx 
```shell

screen - detach
---------------

```shell
ctrl-shift a-d
```

```shell
tmux attach-session

-------------

```shell
ctrl b d
